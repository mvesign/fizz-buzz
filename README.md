# Fizz Buzz

Small Fizz Buzz program created using TDD. Following set of rules was the only thing that was given:

> Write a program that prints the numbers from 1 to 100. If it’s a multiple of 3, it should print “Fizz”. If it’s a multiple of 5, it should print “Buzz”. If it’s a multiple of 3 and 5, it should print “Fizz Buzz”.

### How to use this solution

###### 1. Open solution in Visual Studio
```PowerShell
devenv.eve FizzBuzz.sln
```

###### 2. Build solutions
```
[F6] / [Ctrl]-[Shift]-[B]
```

###### 3. Run application
```
[F5]
```

###### 4. Check output
